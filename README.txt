지니짱 감사일기 · v2.7.2-hotfix-modal-font-icons

이 패치는 ‘로그인 모달이 안 닫힘 / 버튼 반응 없음 / 글씨체·아이콘 깨짐’을 바로잡는 응급 안정화 버전입니다.
요청하신 것만 손봤고, 나머지는 바꾸지 않았습니다.

적용 방법
1) 이 폴더의 `index.html`, `styles.css`, `app.js`를 GitHub Pages 루트에 덮어쓰기 하세요.
2) Firebase를 쓰는 경우: 기존 firebase SDK와 config 스크립트 태그가 있다면 그대로 유지하세요.
   (없어도 로컬 저장으로 작동합니다.)
3) 배포 후 새로고침(필요시 사파리에서 사이트 데이터 삭제)하면 끝.

포함 내용
- 모달: 닫기/배경터치/ESC 모두 동작, 열릴 때 배경 스크롤 잠금
- 탭: 데일리/위클리/검색/설정 해시 라우팅으로 정상 전환
- 폰트: Gowun Dodum(감성체) 한 벌을 UI/입력/플레이스홀더까지 통일 적용
- 저장: Firestore가 있으면 users/<uid>/emotion/{YYYY-MM-DD}, weekly/{YYYY-Www} 로 저장,
        없으면 localStorage에 안전 저장
- 검색: 카드형 결과 표시(읽기 쉬움)
- 푸터: 버전만 심플 표기

문제 생기면 말씀 주세요. 바로 고쳐서 드릴게요.
